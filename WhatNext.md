# What Next for FileWork?

## Basic Features

### - File interpretation
* SimilarityCheck
  * Reads a file against another and returns a radian value
* FileSegregate.dictBurner
  * Reads the file extensions within a specified folder
* FileAnalytic.open_Files_Find_Empty
  * Reads files within a specified folder and returns the files which are empty
### - Files manipulation
* FileAnalytic.file_cull
  * Allows for systemic removal of files from a folder based on a user specified metric
